Glance Es un programa de codigo abierto desarrollado por Kitware para la visualizacion de imagenes y volumenes 3d con vtk.js. Desde datos medicos hasta estructuras geometricas, todo puede representarse con esta libreria. Se ha usado la app de kitware glance como base del proyecto tal y como se describe en su pagina oficial. De esta manera conseguimos implementar ITK.js (que es la libreria predecesora de VTK.js. ) para la lectura de imagenes medicas.
